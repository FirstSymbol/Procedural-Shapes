# Procedural Shapes for Unity UI (v2.2)

**Procedural Shapes** — это профессиональный инструмент для создания высококачественной векторной графики в Unity UI (uGUI) на основе математики SDF. Позволяет создавать гибкие интерфейсы, иконки и сложные маски без растровых спрайтов.

---

## ✨ Что нового в v2.2

*   **Мгновенный фидбек**: Любые изменения резаков (Boolean Operators) или масок теперь отображаются мгновенно в Editor и Runtime благодаря новой системе уведомлений `OnShapeChanged`.
*   **Умное дублирование**: Исправлена проблема "связанных" объектов. Теперь при дублировании (Ctrl+D) фигуры с булевыми операциями получают уникальные экземпляры материалов автоматически.
*   **Path как Оператор**: Теперь сложные векторные пути можно использовать не только как основную форму, но и как "резак" в списке булевых операций.
*   **Стабильный UI**: Инспектор полностью избавлен от потери фокуса при вводе чисел и "отлипания" слайдеров. Логика `Link Scale` работает плавно.
*   **Быстрое создание**: Добавлено контекстное меню в Hierarchy для мгновенного создания Shape и Shape Mask.

## 🚀 Быстрый старт

1.  **Hierarchy (ПКМ)** -> `UI` -> `Procedural Shapes` -> `Shape`.
2.  В инспекторе выберите **Shape Type**.
3.  Настраивайте форму с помощью **интерактивных ручек (Handles)** прямо в Scene View.

---

## 🎨 Булевы операции (CSG)

Комбинируйте фигуры в реальном времени (до 8 операций на объект):
*   **Типы**: Union (Слияние), Subtraction (Вычитание), Intersection (Пересечение).
*   **Smoothness**: Параметр плавности для создания эффекта "метасфер" (органических соединений).
*   **Иерархия**: Поддерживается вложенность (фигура с вырезами может сама быть резаком для другой фигуры).
*   **Защита**: Встроена защита от циклических зависимостей (StackOverflow).

---

## 📐 Векторные пути (Path)

*   Создание любых форм через точки и кривые Безье.
*   **Режим Closed**: Для заливки сложных контуров.
*   **Режим Stroke**: Для создания линий переменной толщины.
*   **Исправлено**: Границы меша теперь рассчитываются корректно, исключая обрезку пути при выходе за пределы RectTransform.

---

## 🛠 Технические детали

*   **Unique Materials**: Сложные объекты (Booleans, Path, Mask) автоматически используют индивидуальные материалы, простые — эффективно батчатся через `ProceduralMaterialPool`.
*   **SDF Rendering**: Идеальный антиалиасинг (Edge Softness) и бесконечная масштабируемость.
*   **Event System**: Внутренняя система подписок гарантирует, что если вы измените родительский объект или резак, все зависимые фигуры обновятся автоматически.

---

### Устранение проблем

*   **Курсор прыгает при вводе**: Обновите плагин до v2.2. Эта проблема UI Toolkit была исправлена через оптимизацию Binding-системы.
*   **Объект обрезается**: Проверьте параметр `Edge Softness` или убедитесь, что в режиме `Path` все точки находятся в пределах разумного диапазона (меш расширяется автоматически).
*   **Не работает маска**: Убедитесь, что на целевом `Image` есть компонент `Soft Maskable Image`.
