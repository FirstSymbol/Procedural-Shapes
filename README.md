# Procedural Shapes for Unity UI (v2.1)

**Procedural Shapes** — это профессиональный инструмент для создания высококачественной векторной графики в Unity UI (uGUI) на основе математики Signed Distance Fields (SDF). Плагин позволяет создавать гибкие и масштабируемые интерфейсы, иконки, сложные маски и эффекты без использования растровых спрайтов. В версии 2.1 добавлен полностью переработанный инспектор на **UI Toolkit** и улучшенные визуальные ручки для Scene View.

## ✨ Основные возможности

*   **Новый Modern UI**: Инспектор полностью переписан на UI Toolkit (Figma-style дизайн, динамические списки, мгновенный отклик).
*   **Расширенные примитивы**: Прямоугольник (Squircle/iOS), Эллипс, Многоугольник, Звезда, Капсула, Линия, Кольцо, Треугольник, Сердце.
*   **Векторные пути (Path)**: Создание любых форм через ломаные линии и кривые Безье.
*   **Интерактивный Scene Editor**: Редактирование скруглений, градиентов и узлов вектора прямо во Viewport с кастомными цветными ручками.
*   **Идеальный рендеринг (SDF)**: Бесконечное разрешение и идеальное сглаживание краев (AA).
*   **Булевы операции (CSG)**: Union, Subtraction, Intersection с поддержкой плавного слияния (Metaballs).
*   **Стилизация и Эффекты**: Drop Shadow, Inner Shadow, Outer Glow, Inner Glow, Stroke, Blur, Bevel (Fake 3D).
*   **Оптимизация (Batching)**: Система Material Pooling объединяет идентичные фигуры в один Draw Call.

## 🚀 Быстрый старт

1.  В Unity перейдите в меню: `GameObject -> UI -> Procedural Shapes -> Shape`.
2.  В инспекторе выберите **Shape Type**.
3.  Используйте **цветные интерактивные ручки (Handles)** в окне Scene для настройки формы.

---

## 🎨 Интерактивное управление (Scene View UX)

Редактор на сцене стал интуитивно понятнее благодаря цветовому кодированию ручек:

*   **Циановые ручки (Cyan)**: Изменяют радиус скругления углов прямоугольника.
*   **Зеленые ручки (Green)**: Управление основными точками векторного пути.
*   **Желтые/Золотые (Yellow)**: Настройка контрольных точек Безье, параметров кольца и градиента.
*   **Оранжевые (Orange)**: Изменение внутреннего радиуса звезд.

---

## 📐 Векторные пути (Path)

Создание сложных кастомных форм:
1.  Установите `Shape Type` -> `Path`.
2.  Добавляйте точки в массив или кликайте прямо в Scene View.
3.  Каждая точка может быть **Line** (прямая) или **Bezier** (кривая).
4.  Режим **Closed** для заливки или разомкнутый контур с настройкой толщины.

## ✂️ Булевы операции и Маски

### Boolean Operations
Комбинируйте фигуры в реальном времени. Поддерживается до 8 операций с параметром `Smoothness` для эффекта "метасфер".

### Alpha-Aware Soft Masking
Компонент **`ProceduralShapeMask`**:
*   Учитывает прозрачность градиента маски.
*   Поддерживает сложные вырезы через булевы операции.
*   Работает со стандартными `Image` (через скрипт `ProceduralSoftMaskable`).

## 🛠 Технические детали

*   **Material Pooling**: Автоматическое кэширование материалов для максимального батчинга.
*   **GPU SDF**: Вычисления в шейдере для мгновенной отрисовки любой сложности.
*   **CPU Raycasting**: Фильтрация кликов точно по контуру фигуры.
*   **Gradient Atlas**: Динамическое запекание градиентов в текстурный атлас (512px).

---

### Устранение частых проблем

*   **Белая заливка**: Проверьте, не переполнен ли атлас градиентов.
*   **Маска не обрезает Image**: Убедитесь, что на объекте с Image есть компонент `Soft Maskable Image`.
*   **Интерфейс не обновился**: При добавлении элементов (эффекты/булевы) инспектор обновляется автоматически. Если этого не произошло, перевыделите объект.
