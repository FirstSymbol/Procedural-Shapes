# Procedural Shapes for Unity UI (v2.0)

**Procedural Shapes** — это профессиональный инструмент для создания высококачественной векторной графики в Unity UI (uGUI) на основе математики Signed Distance Fields (SDF). Плагин позволяет создавать гибкие и масштабируемые интерфейсы, иконки, сложные маски и эффекты без использования растровых спрайтов. В версии 2.0 добавлена поддержка произвольных путей, полноценный редактор на сцене и продвинутая оптимизация материалов.

---

## ✨ Основные возможности

*   **Расширенные примитивы**: Прямоугольник (Squircle/iOS), Эллипс, Многоугольник, Звезда, Капсула, Линия, Кольцо, **Треугольник**, **Сердце**.
*   **Векторные пути (Path)**: Создание абсолютно любых форм через ломаные линии и **кривые Безье**.
*   **Интерактивный Scene Editor**: Редактирование скруглений, градиентов и узлов вектора прямо во Viewport (как в Figma или Illustrator).
*   **Идеальный рендеринг (SDF)**: Бесконечное разрешение и идеальное сглаживание краев (AA) при любом масштабировании.
*   **Булевы операции (CSG)**: Union, Subtraction, Intersection с поддержкой Metaballs (плавное слияние).
*   **Стилизация и Эффекты**: Drop Shadow, Inner Shadow, Outer Glow, **Inner Glow**, Stroke, Blur, **Bevel (Fake 3D)**.
*   **Заливка и Паттерны**: Сплошной цвет, Линейный, Радиальный и Угловой градиенты, заливка текстурным паттерном.
*   **Альфа-маскирование**: Полная поддержка маскирования дочерних элементов с учетом прозрачности заливки и градиента маски.
*   **Оптимизация (Batching)**: Система **Material Pooling** объединяет идентичные фигуры в один Draw Call, сохраняя идеальный Canvas Batching.

---

## 🚀 Быстрый старт

1.  В Unity перейдите в меню: `GameObject -> UI -> Procedural Shapes -> Shape`.
2.  В инспекторе выберите **Shape Type**.
3.  Для редактирования формы используйте **интерактивные ручки (Handles)** прямо в окне Scene.

---

## 🎨 Интерактивное управление (Scene View UX)

В версии 2.0 большинство настроек вынесено в окно Scene для максимально быстрого прототипирования:

*   **Corner Radii**: Тяните за точки внутри углов прямоугольника, чтобы менять радиус скругления каждого угла независимо.
*   **Gradient Edit**: Перемещайте центральную точку градиента для смещения (`Offset`) или крайнюю точку для вращения (`Angle`) и масштабирования (`Scale`).
*   **Path Editor**: При выборе `ShapeType.Path` вы можете двигать узлы, вытягивать "усы" Безье и создавать сложные кривые.

---

## 📐 Векторные пути (Path)

Новый тип геометрии позволяет создавать сложные кастомные формы:
1.  Установите `Shape Type` -> `Path`.
2.  Добавляйте точки в массив или используйте Scene View для визуального построения.
3.  Каждая точка может быть **Line** (прямая) или **Bezier** (кривая с контрольными точками).
4.  Поддерживаются режимы **Closed** (замкнутый контур с заливкой) или разомкнутая ломаная линия с настраиваемой толщиной.

---

## ✂️ Булевы операции и Маски

### Boolean Operations
Комбинируйте фигуры, используя другие `ProceduralShape` как операнды. Поддерживается до 8 вложенных операций с параметром `Smoothness` для создания эффекта капель (Metaballs).

### Alpha-Aware Soft Masking
Компонент **`ProceduralShapeMask`** превращает фигуру в маску. В отличие от стандарта, наши маски:
*   Учитывают прозрачность градиента (объекты плавно исчезают).
*   Учитывают булевы вырезы в маске.
*   Работают со стандартными `Image` (через скрипт `ProceduralSoftMaskable`).

## 🖌 Новые эффекты и фичи

*   **Bevel (Fake 3D)**: Создает эффект объема и освещения (блики и тени), превращая 2D шейп в выпуклую кнопку.
*   **Inner Glow**: Мягкое свечение по внутреннему контуру фигуры.
*   **Dashed Stroke**: Поддержка пунктирных и штриховых линий для эффекта обводки (настраивается `Dash Size` и `Dash Space`).
*   **Triangle & Heart**: Новые математически точные примитивы треугольника и сердца.

---

## 🛠 Технические детали и Оптимизация

*   **Material Pooling**: Плагин автоматически кэширует материалы. Если на сцене 100 кнопок с одинаковыми параметрами (цвет, скругление, маска), они будут отрисованы **одним Draw Call**.
*   **GPU SDF**: Вся математика выполняется в шейдере, что обеспечивает мгновенную скорость отрисовки даже очень сложных форм.
*   **CPU Raycasting**: Фильтр кликов вычисляет SDF на процессоре, обеспечивая попадание точно по контуру фигуры (игнорируя прозрачные зоны).
*   **Gradient Atlas**: Все градиенты запекаются в единый атлас (512px) с автоматической защитой от переполнения и стабильным хэшированием.

---

### Устранение частых проблем

*   **Белая заливка вместо градиента**: Убедитесь, что атлас не переполнен. Если проблема сохраняется, измените любой параметр градиента для принудительного обновления.
*   **Маска не обрезает Image**: Проверьте наличие компонента `ProceduralSoftMaskable` на объекте с `Image`.
*   **Сломанный батчинг**: Батчинг работает только для объектов с абсолютно идентичными параметрами (включая маски и булевы операции).
