# Procedural Shapes for Unity UI

**Procedural Shapes** — это мощный инструмент для создания векторной графики в Unity UI (uGUI) с использованием полей расстояний (SDF). Плагин позволяет создавать сложные интерфейсы, иконки и эффекты без использования спрайтов, с поддержкой булевых операций в реальном времени, градиентов, теней и мягких масок.

---

## ✨ Основные возможности

*   **Примитивы**: Прямоугольник (с закруглением), Круг, Многоугольник, Звезда.
*   **SDF Рендеринг**: Идеально гладкие края при любом масштабе и зуме.
*   **Булевы операции**: Объединение, Вычитание, Пересечение и XOR. Поддержка вложенности и сглаживания (Metaballs).
*   **Эффекты**: Тень (Drop Shadow), Внутренняя тень (Inner Shadow), Обводка (Stroke), Размытие (Blur).
*   **Мягкие маски**: Обрезка изображений и фигур по сложной форме с учетом прозрачности и градиентов.
*   **Производительность**: Все вычисления происходят в шейдере (GPU).

---

## 🚀 Быстрый старт

1.  В Unity перейдите в меню: `GameObject -> UI -> Procedural Shapes -> Shape`.
2.  На сцене появится объект с компонентом `ProceduralShape`.
3.  В инспекторе вы можете изменить **Shape Type** (тип фигуры), радиусы скругления и цвета.

---

## 🎨 Настройка Фигуры (Geometry)

### Типы фигур (Shape Type)
*   **Rectangle**: Прямоугольник. Поддерживает независимое скругление каждого угла (`Corner Radii`) и сглаживание углов (`Corner Smoothing` / Squircle).
*   **Ellipse**: Круг или эллипс.
*   **Polygon**: Правильный многоугольник. Настраивается количество сторон (`Sides`), скругление углов (`Rounding`) и поворот.
*   **Star**: Звезда. Настраивается количество лучей, соотношение радиусов (`Ratio`), скругление внешних и внутренних углов.
*   **None**: Пустая фигура. Полезна как контейнер для булевых операций (когда нужно собрать форму только из добавленных фигур).

### Общие параметры
*   **Shape Scale**: Масштаб фигуры внутри `RectTransform`. `1.0` — фигура занимает всю область.
*   **Edge Softness (AA)**: Мягкость краев (Антиалиасинг). `0` — жесткие пиксельные края, `1+` — мягкие края.

## ✂️ Булевы операции (Boolean Operations)

Вы можете комбинировать фигуры, используя другие объекты `ProceduralShape` как операнды.

### Как использовать:
1.  Создайте основную фигуру (например, **Shape A**).
2.  Создайте вторую фигуру (например, **Shape B**), настройте её размер и положение.
3.  Выберите **Shape A**. В разделе `Boolean Operations` нажмите `+`.
4.  Перетащите **Shape B** в поле `Source Shape`.
5.  Выберите тип операции (например, `Subtraction`).

### Типы операций:
*   **Union**: Объединяет фигуры.
*   **Subtraction**: Вырезает вторую фигуру из первой.
*   **Intersection**: Оставляет только общую область.
*   **XOR**: Исключающее ИЛИ (удаляет область пересечения).

### Настройки операции:
*   **Smoothness**: Позволяет делать плавные переходы между фигурами (эффект "капли" или метаболов).
*   **Порядок**: Используйте кнопки `▲` и `▼` в инспекторе, чтобы менять порядок применения операций.

### 💡 Советы:
*   **Disable Rendering**: Если вы используете фигуру только как "нож" (для вырезания), включите галочку `Disable Rendering` на самом объекте-ноже. Он исчезнет визуально, но продолжит работать в булевой операции.
*   **Вложенность**: Вы можете вычесть Фигуру C из Фигуры B, а результат вычесть из Фигуры A. Система поддерживает сложную иерархию.

## 🖌 Эффекты (Effects)

В разделе `Effects` можно добавлять слои стилизации. Порядок эффектов важен (отрисовываются снизу вверх).

1.  **Drop Shadow**: Падающая тень.
    *   *Offset*: Смещение тени.
    *   *Blur*: Размытие тени.
    *   *Spread*: Расширение тени.
2.  **Inner Shadow**: Внутренняя тень. Работает аналогично, но внутри фигуры.
3.  **Stroke**: Обводка.
    *   *Width*: Толщина.
    *   *Alignment*: Положение (Inside, Center, Outside).
4.  **Blur**: Размытие самой фигуры. Превращает фигуру в "пятно" или свечение.

Все эффекты поддерживают градиентную заливку.

## 🎭 Система Масок (Soft Masking)

Плагин включает мощную систему масок, которая поддерживает полупрозрачность, градиенты и булевы операции.

### 1. Создание Маски
Добавьте компонент `ProceduralShapeMask` на любой объект с `ProceduralShape`.
*   Этот объект станет маской.
*   Параметр **Softness** регулирует мягкость краев маски.

### 2. Маскирование Фигур (Procedural Shapes)
Любой **дочерний** объект с компонентом `ProceduralShape` автоматически распознает маску родителя.
*   Дополнительных компонентов не требуется.
*   Фигура будет обрезана по форме родителя (с учетом всех его булевых вырезов).
*   Прозрачность родителя (альфа-канал градиента) также применяется к ребенку.

### 3. Маскирование Картинок (UI Images/RawImages)
Чтобы замаскировать обычную картинку (Unity Image):
1.  Поместите Image внутрь объекта-маски.
2.  Добавьте на Image компонент **`ProceduralSoftMaskable`**.
3.  Картинка будет обрезана по сложной форме процедурной маски.

---

## 🛠 Технические детали

*   **Shader**: `UI/ProceduralShapes/Shape` и `UI/ProceduralShapes/SoftMaskedImage`.
*   **Лимиты**: Текущая реализация поддерживает до **8** активных булевых операций на одну фигуру.
*   **SDF**: Используется Signed Distance Field математика для бесконечного разрешения.

---

### Устранение проблем

*   **Маска не работает на Image**: Убедитесь, что на Image висит компонент `ProceduralSoftMaskable` и он является потомком объекта с `ProceduralShapeMask`.
*   **Фигура исчезла при добавлении в булеву операцию**: Убедитесь, что вы не пытаетесь применить фигуру саму к себе.
*   **Желеобразный эффект при движении**: Исправлено в версии 1.0. Матрицы трансформации теперь учитывают Pivot и Scale корректно.
